# Installation Guide - Bitcoin Wallet Generator

This guide will walk you through installing and using the Bitcoin Wallet Generator ViolentMonkey script.

## Step 1: Install a Userscript Manager

Choose one based on your browser:

### Google Chrome / Microsoft Edge / Brave
1. Open Chrome Web Store
2. Search for "ViolentMonkey" or "TamperMonkey"
3. Click "Add to Chrome" / "Add to Edge"
4. Confirm the installation

**Recommended:** [ViolentMonkey](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag)

### Mozilla Firefox
1. Open Firefox Add-ons
2. Search for "ViolentMonkey" or "TamperMonkey"
3. Click "Add to Firefox"
4. Confirm the installation

**Recommended:** [ViolentMonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)

### Safari (macOS/iOS)
1. Open App Store
2. Search for "Userscripts"
3. Download and install the app
4. Enable the extension in Safari preferences

## Step 2: Install the Bitcoin Wallet Generator Script

### Method 1: Direct Installation (If Available)
1. Click on the raw link for `bitcoin-wallet-generator.user.js`
2. Your userscript manager will automatically detect it
3. Click "Install" or "Confirm Installation"
4. Done!

### Method 2: Manual Installation
1. Download or copy the contents of `bitcoin-wallet-generator.user.js` from this repository
2. Click on your userscript manager icon (usually in the browser toolbar)
3. Select "Create a new script" or click the "+" button
4. Delete any default content in the editor
5. Paste the entire script code
6. Click "Save" (or press Ctrl+S / Cmd+S)
7. Close the editor

## Step 3: Verify Installation

1. Navigate to any webpage (e.g., https://google.com)
2. Look for a purple panel in the top-right corner labeled "ü™ô Bitcoin Wallet Generator"
3. If you see it, the installation was successful!

## Step 4: First Use

### Generate Your First Wallet
1. Click the **"Generate New Wallet"** button in the panel
2. Wait a moment while the wallet is generated
3. You'll see:
   - Your 24-word backup code (mnemonic)
   - Legacy Bitcoin address (starts with "1")
   - SegWit Bitcoin address (starts with "bc1")
   - Private key (WIF format)
   - Public key

## Step 5: Import to TrustWallet

### On Mobile (TrustWallet App)
1. Download TrustWallet from:
   - [iOS App Store](https://apps.apple.com/app/trust-crypto-bitcoin-wallet/id1288339409)
   - [Google Play Store](https://play.google.com/store/apps/details?id=com.wallet.crypto.trustapp)

2. Open TrustWallet

3. Select "I already have a wallet"

4. Choose "Import using recovery phrase"

5. Enter your 24-word backup code (one word per field)
   - Make sure to enter words in the correct order
   - Check for typos carefully

6. Set up your security (PIN, biometrics)

7. Your wallet is now imported!

### Verify Import
1. In TrustWallet, tap on Bitcoin
2. Tap "Receive"
3. Compare the displayed address with the one generated by the script
4. They should match exactly (you can check both Legacy and SegWit addresses)

## Troubleshooting

### Script Panel Not Showing
**Problem:** Can't see the purple panel after installation

**Solutions:**
1. Refresh the page (F5 or Ctrl+R)
2. Check that the userscript manager is enabled:
   - Click the extension icon
   - Ensure the script is enabled (toggle should be ON)
3. Clear browser cache and reload
4. Check browser console for errors:
   - Press F12
   - Look for error messages in red

### Required Libraries Not Loading
**Problem:** Error messages about missing libraries (bip39, bitcoinjs-lib, etc.)

**Solutions:**
1. Check your internet connection
2. Disable any ad-blockers temporarily
3. Allow the script to access CDN resources:
   - In userscript manager settings
   - Enable external script loading

### Balance Check Not Working
**Problem:** Balance check fails or shows error

**Solutions:**
1. Verify you've generated a wallet first
2. Check internet connection
3. APIs may have rate limits - wait a few minutes and try again
4. Check browser console (F12) for specific error messages

### Copy Buttons Not Working
**Problem:** Can't copy text to clipboard

**Solutions:**
1. Ensure page is loaded over HTTPS (browsers restrict clipboard on HTTP)
2. Manually select and copy the text:
   - Click and drag to select
   - Press Ctrl+C (Windows/Linux) or Cmd+C (Mac)
3. Grant clipboard permissions when prompted

### Script Conflicts
**Problem:** Script doesn't work on certain websites

**Solutions:**
1. The script runs on all websites by default
2. Some websites may have strict Content Security Policies
3. Try on a different website
4. Check userscript manager logs for blocked requests

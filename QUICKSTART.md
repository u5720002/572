## Quick Troubleshooting

**Panel not showing?**
- Refresh page (Ctrl+F5)
- Check script is enabled in script manager

**Searches not starting?**
- Make sure you're on bing.com
- Check if you're logged into Microsoft account

**Points not appearing?**
- Wait 5-15 minutes
- Visit [rewards dashboard](https://rewards.microsoft.com/)
